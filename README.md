# Magic-Jukebox
A program that is intended to play music from your computer randomly, using VLC running in RC mode to fulfill its purpose.

Project Name: Magic Jukebox

Author: wktdlb19

Start Date: Approximately February 10, 2026

End Date: April 30, 2026

Compatibility: Windows 10 and 11 (Compatibility with older operating systems such as 8.1 and 7 is unknown, as they have not been tested)

Synopsis: Magic Jukebox is a program developed in AutoHotkey v2 that is intended to play music from your computer randomly, using VLC running in RC mode to fulfill its purpose. It includes several features, among them: skipping forward and backward between songs, adding or removing folders designated to contain music, playing specific songs through a built-in search tool, a playlist creation and playback system, 23 languages to choose from, a customizable keyboard shortcut system, and a built-in mini player capable of using most functions, complete with a volume bar and a progress bar for the current song, as well as artist, album and track information for whatever is currently playing.

Important notes:

-Magic Jukebox was developed entirely (in terms of code) with the help of artificial intelligence, commonly known as Vibe Coding. ChatGPT was used to initialize the project, but 80% of the features were implemented using Claude.

-The UI panels, design ideas and functionalities were planned by its creator, wktdlb19. Things like the mini player, the icons, and other assets were designed by him in Photopea.

-VLC (VideoLAN) is required to access all the features the program relies on.

-The following formats are supported: mp3|flac|wav|ogg|m4a|aac|opus|wma|aiff|mp4|webm, as well as shortcut files pointing to files with the aforementioned extensions. If a format supported by VLC is missing, it can be manually added by editing the uncompiled source file.

-When working with the uncompiled project, AutoHotkey v2 is required to open it (Magic Jukebox.ahk) and to compile it using Ahk2exe.

-Upon launch, the program hides itself in the system tray. The user must right-click the icon to open the options menu.

-Left-clicking the icon only has one function: playing the next song.

-It is recommended to drag the program's icon from the system tray to the taskbar (alongside the volume and Wi-Fi icons) so that it remains visible and is more convenient to use.

-Magic Jukebox does not use any resource that requires attribution of any kind. All assets are free-license resources found online (see rocola.wav, an edited version of a free-to-use cassette player audio).
